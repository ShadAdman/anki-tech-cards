# KotlinFalsh Anki Cards

This repository contains importable Anki flashcards for my LinkedIn **KotlinFlash series**.

Each file is a single flashcard in Anki-compatible comma-separated format.

## How to use?

1. Open a card file from the repository
2. Download the `.txt` file
3. Open **AnkiDroid**
4. Import -> Text file
5. Select comma as separator
